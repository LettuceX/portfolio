MITS 11: Enterprise UX/UI Modernization

Role: Lead UX Designer Focus: Data Visualization, Workflow Automation, Cognitive Load Management
Tools: Figma, Adobe XD, User Interviews, Design Systems

Design Challenge

Battery testing involves monitoring hundreds of concurrent high-frequency data streams (Voltage, Current, Temperature) across distributed hardware. The legacy desktop interface was cluttered and tabular, causing high cognitive load for engineers trying to identify critical hardware failures in real-time.

1. Information Architecture: The Macro-Micro Pattern

My primary goal was to reduce the "Time to Insight" for operators monitoring large-scale labs. I restructured the application into a tiered information hierarchy to manage data density:

Macro View (Aggregate Dashboard)

A high-level "Command Center" dashboard that aggregates status across 100+ channels. I utilized "Management by Exception" principles—using color (Green/Red/Amber) to draw attention solely to anomalies, allowing operators to ignore healthy systems.

Micro View (Granular Analysis)

Drill-down views for specific channels. Prioritizes real-time telemetry graphs and interactive scrubbing, enabling engineers to perform root-cause analysis on specific hardware events.

2. No-Code Workflow Automation (Schedule Editor)

Engineers needed a flexible way to program complex testing logic (Loops, Conditional Branching) without writing raw script code. I designed a Drag-and-Drop Visual Editor that visualizes the test procedure as a linear timeline.

  • Visual Validation: Steps are represented as modular blocks that snap together. I designed validation logic that visually highlights invalid connections (error states) immediately, preventing runtime errors.
  • Contextual Configuration: Clicking a block reveals a context-sensitive properties panel, keeping the main canvas clean and focused on the logic flow rather than parameter clutter.

3. Atomic Design System

To ensure consistency across the web platform, I established a comprehensive Design System in Figma. This atomic design approach improved development velocity and maintained visual unity across disparate modules.

  • Component Library: Standardized interactive elements (Inputs, Toggles, Dropdowns) with clearly defined hover, active, and disabled states.
  • Accessible Data Viz: Created a semantic color palette specifically for high-density charts, ensuring distinct line visibility and high contrast for accessibility compliance.

4. User-Centered Research & Iteration

The design process was iterative, driven by direct feedback loops with internal hardware engineers.

  • Pain Point Discovery: User interviews revealed that engineers frequently lost mental context of which "Virtual Channel" corresponded to which "Physical Rack Unit."
  • Spatial Mapping Solution: Introduced a "Channel Map" visualization—a graphical grid mirroring the physical rack layout—allowing users to select units spatially, reducing selection errors.

5. Resource Allocation & Orchestration

A critical challenge was enabling users to bundle channels from different physical machines into a single "Test Task." I designed a centralized Resource Management interface.

  • Drag-and-Drop Provisioning: Users can drag available channels from a global pool into a "Task Bucket," visually grouping disparate hardware resources into a single logical unit.
  • Conflict Resolution UI: If a user selects a channel that is currently occupied, the UI provides a clear "Reserved" state with a tooltip estimating availability, preemptively resolving scheduling conflicts.

6. Diagnostic Reporting & QC

For the Quality Control pipeline, I designed a comprehensive "Validation Report" view that transforms raw data into an actionable Pass/Fail verdict.

  • Root Cause Visualization: A specialized view that pinpoints exactly why a battery failed QC (e.g., highlighting specific timestamps where Voltage dropped below threshold).
  • Benchmark Overlays: Designed a comparative feature allowing users to overlay current results against a "Golden Sample" (ideal benchmark), facilitating instant visual quality assurance.
